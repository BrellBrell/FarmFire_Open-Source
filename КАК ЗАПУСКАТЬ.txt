
КАК ЗАПУСТИТЬ?
1. В MongoDB создать БД farmfire-app

2. В БД создать коллекцию servers и импортировать из папки json/farmfire.servers.json

3. В БД создать коллекцию versions и импортировать из папки json/farmfire.versions.json

4. Открыть этот каталог в VS Code.

5. В терминале 1: 
        cd serverHF
        npm i
        npm start
   В терминале 2: 
        cd mainServer
        npm i
        npm start

   В терминале 3: 
        cd farmfire-app
        npm i
        npm run dev


ЗАЧЕМ ДВА СЕРВЕРА?
serverHF можно дублировать много раз и размещать где-угодно, он является исполнителем.
mainServer - главный сервер. mainServer общается со всеми исполнительнными серверами и проверяет входные данные от клиента, чтобы 
исполнительные сервера не падали. Ещё он имеет доступ к БД и взаимодействует с ним. Сервер единственный и центральный.
